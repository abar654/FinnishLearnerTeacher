Reading Resources

Reading is probably the easiest skill to practice when learning a language. There are also so many ways to practice that are fun and interesting and you can do it in almost any location at any time.

Below you will find recommendations for a variety of resource types. Each resource type has examples which are labelled based on what learning level they are appropriate for:
  • Complete Beginner (A0-A1)
  • Beginner (A1-A2)
  • Intermediate (A2-B1)
I have also given each resource a star rating (out of 3) based on how useful it is for this particular skill (but keep in mind the same resource may have a different rating for other skills).
Baby Books ~ Vauvakirjat
Try to match the words with the pictures. Baby books are good for this because words are often in perusmuoto (basic form) and sentences are not long.
Public Libraries
If you are in Finland you will find public libraries in every city. If you are in another country try a Finnish mission or cultural center.
